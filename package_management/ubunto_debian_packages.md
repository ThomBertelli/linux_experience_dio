Packge managers:
`apt-get`
`apt`

List intalled programs:
`apt list --installed`

List upgradeable programs:
`apt list --upgradeable`

Search available programs:
`apt search <program name>`

Install program:
`apt install <program name>`

Unistall program:
`apt remove <program name> -y `
"-y" -> confirm option

Edit packages sources:
`apt edit-sources`
